# Cadastro POO - DGT2821

Trabalho da disciplina Programação Back-end com Java.
O objetivo foi implementar um sistema de cadastro em modo texto com persistência em arquivos binários, utilizando Java puro (sem frameworks) e o padrão DAO/Repository.

## Tecnologias
- Java (JDK)
- NetBeans (Ant)
- Persistência em Arquivo (.bin)

## Documentação
O relatório completo solicitado pela disciplina está em PDF na pasta de documentação:
[Acessar Relatório em PDF](./docs/Relatorio_Pratica_DGT2821.pdf)

## Como rodar
1. Clone o repositório.
2. Abra o projeto no NetBeans.
3. Execute a classe CadastroPOO no pacote cadastropoo.
